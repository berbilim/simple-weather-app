# Simple Weather App

Weather API Flask Application
This repository contains a simple Flask application that fetches weather data from the OpenWeatherMap API and returns the current temperature and weather description for a specified city.

Features
Retrieves weather data for a specified city.
Returns current temperature in Celsius.
Returns weather description.
Handles various exceptions and returns appropriate error messages.
